# 🚗 EV Adoption Analytics

## Overview

This project analyzes consumer behavior and factors influencing Electric Vehicle (EV) adoption using a dataset of 50,000 respondents. The objective is to identify the key drivers and barriers affecting EV adoption and generate actionable business insights through data analytics techniques.

## Problem Statement

Electric Vehicle adoption is growing worldwide, but consumer acceptance varies based on economic, demographic, and behavioral factors. This project aims to understand which factors have the greatest impact on EV adoption likelihood.

## Objectives

* Perform data cleaning and preprocessing.
* Analyze demographic and behavioral patterns.
* Identify factors influencing EV adoption.
* Generate business insights and recommendations.
* Visualize trends using statistical and exploratory analysis.

## Dataset Features

* Age
* Annual Income
* Education Level
* City Type
* Daily Commute Distance
* Current Vehicle Type
* Fuel Expense Per Month
* Charging Station Accessibility
* Environmental Awareness Score
* Technology Affinity Score
* Range Anxiety Score
* EV Knowledge Score
* EV Adoption Likelihood

## Analysis Performed

* Data Cleaning
* Missing Value Treatment
* Exploratory Data Analysis (EDA)
* Univariate Analysis
* Bivariate Analysis
* Correlation Analysis
* Consumer Segmentation
* Business Insight Generation

## Key Findings

* Consumers with higher income show greater EV adoption likelihood.
* Environmental awareness positively influences EV adoption.
* Home charging availability increases adoption rates.
* Range anxiety is a significant barrier to EV adoption.
* EV knowledge and technology affinity improve adoption probability.
